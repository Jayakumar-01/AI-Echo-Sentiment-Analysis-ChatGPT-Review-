# AI Echo – Sentiment Analysis Project

This project performs sentiment analysis on app reviews using Natural Language Processing and Machine Learning.

## Features
- Data preprocessing and cleaning
- Sentiment labeling
- TF-IDF text vectorization
- Logistic Regression model training
- Interactive Streamlit web application

## Project Structure
- `data/` → Dataset files
- `models/` → Trained model and vectorizer
- `notebooks/` → Python scripts for each pipeline step
- `app/` → Streamlit frontend
- `reports/` → Output screenshots

## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt

Step : 2 Run Streamlit app:
